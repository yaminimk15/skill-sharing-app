# Skill Exchange Web Application

## 📌 Overview

This is a Java full-stack web application that allows users to share skills, request sessions, and interact with other users.

## 🚀 Features

* User Registration & Login
* Skill Posting and Browsing
* Session Request System
* Accept/Reject Requests
* Notifications
* Rating & Feedback System

## 🛠️ Tech Stack

* Java (Servlets, JSP)
* HTML, CSS
* MySQL

## ⚙️ How to Run

1. Clone the repository
2. Import into Eclipse
3. Configure MySQL database
4. Run on Apache Tomcat server
